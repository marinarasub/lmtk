\doxysubsubsection{Detailed Description}
Represents an rectangular, colored image. 

\mbox{\hyperlink{classimage_1_1_image}{Image}} is of size width $\ast$ height (OR x $\ast$ y OR \#columns $\ast$ \#rows). Stores color/ pixel data in RGBA format and provides basic image creation and modification functions.

\begin{DoxyNote}{Note}
The image class does not provide I/O (read/write) functions as it is just a respresentation in memory. For such, use a class for a well defined image file format. 
\end{DoxyNote}
