\doxysubsection{Detailed Description}
Struct for pixel data as a 1D array in memory where each row is stored consecutively, top to bottom. 

The image has a defined width and height, and such a pixel at location (x, y) is stored at index {\ttfamily y $\ast$ width() + x}.
