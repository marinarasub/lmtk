\doxysubsubsection{Detailed Description}
Class for box blur. 

A box blur convolution is all 1\textquotesingle{}s, smoothing the image with equally weighted surrounding pixels. Repeated iterations / multiple passes approximates the Gaussian Blur. \begin{DoxySeeAlso}{See also}
